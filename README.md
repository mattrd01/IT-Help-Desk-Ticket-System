# IT Help Desk Ticket System

## Overview

The IT Help Desk Ticket System is a desktop application built with Python, Tkinter, and JSON. It allows support teams to create, track, and manage customer support tickets through a simple graphical user interface.

## Features

* Create support tickets
* Assign ticket priorities (Low, Medium, High)
* Track ticket status
* View all open tickets
* Mark tickets as resolved
* Delete tickets
* Store ticket data using JSON

## Technologies Used

* Python
* Tkinter
* JSON

## How to Run

1. Install Python 3.
2. Download the project files.
3. Open a terminal in the project directory.
4. Run:

python help_desk_ticket_system.py
